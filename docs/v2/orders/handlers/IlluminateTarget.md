# ILLUMINATE_TARGET

Manually illuminates a target with a fire-control radar or laser. This is required for some semi-active weapons to guide correctly.

## Parameters

| Parameter | Type | Description |
| :--- | :--- | :--- |
| `targetId` | `number` | (Optional) Ground-truth ID of the target. |
| `trackId` | `string` | (Optional) Perceived track ID. |
