# Sensor & EW Orders

These orders control the electronic emissions and detection capabilities of a platform.

- **[SET_SENSOR_STATE](./handlers/SetSensorState.md)**: Controls individual sensors (Radar, Sonar, etc.).
- **[SET_MASTER_EMCON](./handlers/SetMasterEmcon.md)**: Changes the platform's global Emission Control (EMCON) level.
- **[SET_MAST_STATE](./handlers/SetMastState.md)**: Controls the extension of masts on submarines.
- **[SET_TOWED_ARRAY](./handlers/SetTowedArray.md)**: Controls the deployment or retrieval of a towed sonar array.
- **[DEPLOY_DECOY](./handlers/DeployDecoy.md)**: Spawns a countermeasure (Chaff/Flare).
- **[DEPLOY_SONOBUOY](./handlers/DeploySonobuoy.md)**: Spawns a consumable sonar sensor.
