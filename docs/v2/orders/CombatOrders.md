# Combat & Tactical Orders

These orders manage targeting, weapon deployment, and engagement rules.

- **[ENGAGE_TARGET](./handlers/EngageTarget.md)**: Locks onto a specific target for automated engagement.
- **[ALLOCATE_WEAPON](./handlers/AllocateWeapon.md)**: Assigns a specific weapon type or quantity to a target.
- **[DROP_TARGET](./handlers/DropTarget.md)**: Clears the unit's current target.
- **[BOL_LAUNCH](./handlers/BolLaunch.md)**: Fires a weapon toward a specific heading without a sensor lock.
- **[ILLUMINATE_TARGET](./handlers/IlluminateTarget.md)**: Manually activates fire-control radars or lasers.
- **[SET_WEAPONS_POSTURE](./handlers/SetWeaponsPosture.md)**: Sets the global firing rule for the entity (FREE, TIGHT, HOLD).
- **[SET_WRA](./handlers/SetWra.md)**: Configures Weapon Release Authorization rules.
